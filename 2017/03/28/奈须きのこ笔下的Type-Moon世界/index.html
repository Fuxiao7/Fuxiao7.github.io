奈須きのこ笔下的型月世界观

2017-03-28

前言

这些是在上一个电脑里整理过的东西，前几天重新整合了一下，添加了Fate/Grand Order里的内容。
写的时候尽量避免了一些有争议的设定。对Type-Moon了解的同学，可以帮我捉个虫。
最开始是写来安利别人的，但是发现安利力度远远不够，果然还是没有笔力的锅。

1、一个根源，两大抑制力

1.1 根源

根源就是月世界的万物本源，是一切事物的存在规则。

做一个类比的话，根源就是道家思想里的道、太极。用电脑网络做比较的话，根源就是生成一切用户数据的根服务器。

1.2 星球抑制力：盖亚

那么说完根源，两种抑制力分别是什么呢。

第一种抑制力被称为盖亚，他是地球上全部生物意识的统合。
我们知道月世界里的许多事物，都是有现实的资料可以参照的。而盖亚这个名字，来源于希腊神话里的大地之母盖亚。

而地球生物意识的统合，这个概念来自于英国科学家提出的盖亚假说。盖亚假说认为，地球是有生命有意识的。这个星球上的一切环境变化，自然灾害，都是地球为了维持自身的存在而做出的变化。

在月世界中，星球的抑制力盖亚，和现实世界中的盖亚假说类似，也是为了防止（抑制）星球的毁灭，而盖亚视人类为星球毁灭的最大威胁。盖亚意志的具体体现，便是《月姬》中出现的“真祖”。盖亚与真祖的关系，我们将在第三部分讲到。

1.3 灵长类抑制力：阿赖耶识

另一种抑制力被称为阿赖耶识，是灵长类生物意识的统合。阿赖耶识这个名词，来自于佛教。而这种灵长类生物意识统合的概念，来自于瑞士心理学家荣格的“人类集体无意识”。这个概念，在另一个日式游戏的著名系列《女神异闻录》中也有体现。

在月世界里，它维持着人类的存续，即防止（抑制）人类的毁灭。为了达到这个目的的具体体现，便是《Fate》系列中的英灵和守护者。英灵、守护者和阿赖耶识的关系，我们会在第四部分谈到。

1.4 两种抑制力矛盾的对立统一

讲到这里，就出现了一对矛盾的对立。

盖亚的目的，是抑制地球的毁灭。为了维持地球存在，它不会管地球上某一具体物种的死活。

阿赖耶识的目的是抑制人类的毁灭，为了这个目的，它不在乎其他物种的死活。毕竟就算地球毁灭，人类还可以选择移民外太空。

这两种抑制力的对立点就在这里。

而矛盾除了对立，也有统一的一面。这种统一就体现在，两种抑制力都要阻止生物接触根源。因为根源是一切事物的本源，一旦有生物接触到根源，接触到世界运行的规则，那他就可能对人类和地球产生毁灭性的威胁。

这是这对矛盾的统一点。

总结，两大抑制力都要阻止人类接触根源，因为接触根源会同时引发星球和人类的毁灭可能，这是二者的共同利益；

但是除开这一点，盖亚是非常热衷于消灭人类的；而阿赖耶识为了保证人类的存续，对星球上其他的物种，也持有“大不了就消灭”的态度。

说完根源和抑制力的概况，我们下面就展开来谈一下，这三种概念，在月世界中的具体体现。

2、根源与起源

根源在月世界中最多的体现，是在《空之境界》这部作品里。

在空之境界中，每个人都有一个“起源”，起源是每个人在根源里的缩影，是人最根本的特性。起源可以是具体事物，也可以是一种概念，它伴随着一个人轮回转世，并且在每一次轮回中，他都会体现出来。

用这个设定举个例子，假设钱钟书的起源是文学，他的一生中，无处不体现着他对文字的敏感和文学的造诣，而和文学相对的数理科学，就与他绝缘，比如数学考试不及格被特招什么的。

我们也可以假设，张小泉，王麻子的起源是刀，因此他们锻制的菜刀品质上乘。
在《空之境界》里，两仪式的起源是空，我们知道，空或者无，是万物的本源。所以两仪式就是那个最接近根源的人。

《Fate/Stay Night》中，卫宫士郎的起源是剑，因此他特别擅长制剑。在Heaven’s Feel线里，士郎起源觉醒，无数的剑从他身体里产生，万剑穿身。

3、盖亚与真祖

在第一部分中，我们谈到了星球抑制力盖亚，与盖亚意志的具体体现：真祖，在本部分中，我们将具体地解释盖亚与真祖的联系。

3.1 真祖

盖亚认定，这个星球上，对星球威胁最大的物种便是人类。因此，作为星球的自卫手段，盖亚创造了最接近人类的真祖来对抗人类。

真祖是盖亚创造的，具有强大力量，不老不死的人形物种，但是真祖有一个缺点，就是需要吸取人类的血液来维持人形。

简单来说，所谓真祖，就是具备一定星球力量的吸血鬼，人形的星球抑制力。
真祖出现在《月姬》《歌月十夜》等作品中，《月姬》女主爱尔奎特公主便是由真祖诞下的“真祖的公主”。

3.2 死徒

真祖为了预防自己无法压抑吸血冲动，而预先准备的人类。真祖通过预先吸取人类的血液，将这些人变成死徒，以便对其进行控制。

真祖与死徒的关系，与其他文学作品中，高等吸血鬼与眷属的关系很像。通过吸血仪式，获得普通人类的控制权，同时赋予普通人类力量，使其成为死徒/眷属。
而这些受控制的死徒中，有一些不甘受困，开始尝试脱离真祖的控制。最开始，有27位死徒摆脱了真祖的控制，被称为“二十七祖”。《Fate》中，远坂家的大师傅，修拜因奥古（“宝石翁”）便是二十七祖之一。

3.3 代行者

代行者并不是盖亚意志的体现，而是人类组织“圣堂教会”为了对抗吸血鬼和死徒而组建的“埋葬机关”的成员。

埋葬机关这个概念，应该使借鉴了欧洲传说中的吸血鬼猎人。

传说，在中世纪前后，吸血鬼和狼人并存，各种怪异出现在欧洲各地，教会成立了专门的机构，用以保护人民。这个机构中的执行者，被称为怪物猎人。
电影《范海辛》将怪物猎人与吸血鬼、狼人的对立关系刻画地淋漓尽致。

3.4 幻想种

在月世界中，魔兽、幻兽、神兽统称为幻想种。即存在与幻想、传说中的物种。幻想种也是盖亚意志的体现，是星球抑制力的一种。幻想种最出名、最顶尖的是龙。

当然，盖亚意志不光通过这种具体的形式体现，也可以通过引发地震、海啸、火山喷发等天灾方式，完成对威胁的消灭，从而保护星球。

4、阿赖耶识与英灵

讲完星球抑制力与它的具体体现，我们来讲一下灵长类抑制力与它的具体体现。
阿赖耶识的具体体现，便是位于英灵之座中的英灵。英灵也是《Fate》系列的核心概念之一。

4.1 英灵

英灵，是对人类历史有重大贡献的人，在死后化为的存在。

英灵代表了人类的希望，是人类信仰的凝结，他们存在于英灵之座中，是守护人类的重要力量。当人类社会面临重大危机时，有能力的人便能通过抑制力召唤英灵，守护人类的希望。

我们可以把英灵理解为人类漫长历史中诞生的无数英雄，他们可以是开疆拓土的帝王，可以是发明伟大技术的科学家，可以是丰富人类精神文明的诗人，一切对人类历史的前进起到积极作用的人，在死后被人们铭记，都可以称为英灵。

在《Fate》中，我们看到，凯尔特神话中的亚瑟王、历史上真实存在的亚历山大大帝、英法百年战争中的吉尔德雷男爵，都作为英灵被降灵魔术召唤。

4.2 守护者

如果说英灵是人类世界的希望，是人类美好愿望的凝结，那守护者就是人类绝望的化身。英灵与守护者虽然都作为灵魂位于英灵之座，都是守护人类的力量，都是阿赖耶识的体现，但守护者通常都是那些死后都无法得到救赎的人的灵魂。

死后化为守护者的人，灵魂供阿赖耶识驱使，完成各种“阻止人类灭亡”的杀人任务，灵魂在不断的杀戮中无法得到解放。

因此，我们说英灵是人类美好希望的化身，守护者是人类绝望的化身。

《Fate》中的英灵卫宫，就是受阿赖耶识支配的守护者。卫宫士郎为了坚持“正义的伙伴”的志向，奔波于世界各地，保护他人。而他生命的终点，竟然是被当成杀人犯，被他保护的人处死。

死后，卫宫士郎与阿赖耶识签订契约，成为守护者，被阿赖耶识驱使，去猎杀“对人类有威胁的人类”，成为了不受自己控制的杀人犯。

4.3 人理存续保障机构：迦勒底

迦勒底是《Fate/Grand Order》中，将人类从既定的毁灭中拯救回来的机构，被称为“人理存续保障机构”。我认为，这是灵长类抑制力的另一种具体体现，不同于英灵、守护者以个体的形式存在，迦勒底是以一个组织的形式存在的灵长类抑制力。

在《Grand Order》中，我们面对的是整个人类文明被抹去的既定事实，唯一存在的只有魔术机构迦勒底，这可以说是人类存在的空前危机，为了在这样绝望的背景下拯救人类，阻止人类文明被抹去，灵长类抑制力给了迦勒底非常强大的支持。迦勒底能够几乎无限制地召唤大量英灵的设备，显然是灵长类抑制力的体现。抑制力给迦勒底无限召唤英灵的能力，让我们的主角回到人类历史上的各个时空特异点，解决人类灭亡的危机。

5、魔术师与魔法使

前面还在说两种抑制力，这里突然以魔术师和魔法使为标题，是为什么呢。

因为型月世界中，存在一种“危险职业”，魔术师。这类人视探求根源为人生的终极目标，在几千年的人类历史中，他们不断尝试接触根源，探求世界的本质。

5.1 探求根源

求真求实看起来并没有错，但是一旦有魔术师到达了根源，那么他就有了从根本上改变世界的力量。这威胁到了星球的存在和人类的存续，因此，他们的这种行为触发了两大抑制力，这两种始终对立的抑制力，在阻止魔术师接触根源这件事上，却是意见统一的、不遗余力的。

5.2 一个现实中的例子

这里，我们可以借用一个完整的现实中的例子作比较。

我们在开头，把根源比作了电脑网络的根服务器。那么，两大抑制力就是装在根服务器上的防火墙。这两个防火墙时常因为兼容性问题打架（在电脑上装两个杀毒软件的后果）。

趁防火墙干架，搞入侵的黑客，就是型月世界中的魔术师，他们想入侵到根服务器，拿到根服务器的数据，之后就可以为所欲为。他们可以给自己的银行账户添几个零，可以让全球股市瞬间崩溃，也可以把各个国家的核弹都弄出来炸掉。
虽然我们不排除有一些个人觉悟很高的黑客，不会改动根服务器的数据，但是总有那么些人，他们会拿数据来搞危险的事情。

这个时候，我们的两大抑制力，也就是防火墙就要出场了。面对各类黑客的入侵行为，两个防火墙一致对外，务求御敌于墙外，不给他们一丝到达根服务器的机会。

根源是世界的本质，两大抑制力相爱相杀，魔术师从中寻找机会，意图接触根源。这就是根源、抑制力与魔术师的关系。

另外，其实在《Fate/Extra》系列中，魔术师真的变成了电脑黑客，到达世界本质的方法，也真的是一台巨大的灵子电脑。

5.3 魔术与魔法

人类在探求根源时，创造的各种改造世界的方法，就是魔术。
当前时间的人类竭尽全力都达不到的改造世界的方法，就是魔法。

能使用魔术的人类，被称为魔术师。而魔术师中，存在极少的几个人，能够使用超越当前人类的力量的魔法，这几人，被称为魔法使。

也就是说，世界上70亿人竭尽一切可能，都做不到的事情，你可以做到，你就是魔法使。

举个例子：
大部分的魔术师，都可以放个大火球，大火球就是魔术。
有极少的那么几个魔术师，他们能移山填海、呼风唤雨，动辄大陆变大洋，这种大家都办不到，只有你能办到的，就是魔法，这样的人，就被称为魔法使。
但是魔术和魔法的界定，也是受时代限制的。移山填海的魔法，随着时间推移，被很多人所掌握，那么，这种魔法，就降格为魔术了。

5.4 五大魔法

在月世界中，世界上只剩下五种魔法，剩下的改造世界的方法，都是魔术。
这五种魔法，被称为“五大魔法”。五大魔法都能到达根源，掌握了五大魔法的魔法使，就是能从根本上影响世界的人。

五大法中的第一、第四法目前还不明确。

而第二法是“平行世界干涉”，即从时空上干涉平行世界的事物，掌握第二法的人，是前文提到的二十七祖之一，修拜因奥古。他可以往来于平行世界，出现在世界的任何地方。

第三法是“灵魂物质化”，又称为天之杯Heavens’s Feel。是将灵魂变为实质的魔法。当灵魂变为物质时，变成物质的个体拥有无尽的魔力和无限的生命，拥有这样特质的个体，已经达到了无所不能的程度，这就是真正的“万能许愿机”。掌握第三法的是爱因兹贝伦家族，但是此魔法后来遗失，为了找回丢失的第三法，爱因兹贝伦家发起了圣杯战争。

第五法可能是时间操纵。准确地说，是操纵时间轴的能力。掌握第五法的是苍崎青子。在《魔法使之夜》中，青子可以将“草十郎的死”从时间轴上切下来，放到无限的未来，再从草十郎的童年，切取10年的时间，补在了时间轴的当下。

5.5 圣杯战争

上面提到了爱因兹贝伦家掌握了第三法，但是后来又遗失掉了。为了找回遗失的第三法，爱因兹贝伦家发起了圣杯战争，圣杯战争就是《Fate》系列的核心。
进行圣杯战争，首先要在一条灵脉上布下“大圣杯”魔术，大圣杯连接灵长类抑制力形成的英灵之座，可以进行英灵的召唤（降灵魔术）。其次，要让被召唤的英灵互相厮杀，当召唤出的英灵在现实世界中死亡后，他们的灵魂会回归英灵之座。最后，最重要的一步，大圣杯这一魔术可以在英灵回归英灵之座时，引导魔术师到达根源。而爱因兹贝伦家，也可以通过这个接触根源的过程，找回丢失的第三法。

这便是整个圣杯战争背后的原理，继续用电脑网络作为比喻的话，那就是：
一群黑客，先从根服务器里拿到一些大家都知道的数据，再通过这些数据，反过来破解防火墙，从而入侵根服务器。

那么，英灵们在被召唤之后，如何保证他们能互相厮杀呢。那就是许给他们“完成愿望”的承诺：只要你们互相厮杀，战至只剩最后一人，圣杯就会出现，完成你的任何愿望。

事实上，在大圣杯魔术接触到根源后，会在布置大圣杯的地方，涌出无尽的魔力，这些魔力足够一个魔术师放出改变世界的魔法。魔法便是从根本上改变世界的方法，也就是“万能的许愿机”。

6、总结

五个部分讲完之后，我们回过头来整体梳理一下。

月世界中，世界存在的根本、万物运行的规则，被成为根源。有这么一群人，穷尽自己的一生，去追求根源、探求世界的本质，这群人被称为魔术师。但是，魔术师接触到根源后，很可能会对世界产生毁灭性的威胁，所以，就有两股抑制力来阻止魔术师接触根源。这两种抑制力，分别是星球的抑制力：盖亚，以及灵长类的抑制力：阿赖耶识。他们分别保护星球的存在和人类的存在。

盖亚为了保护自己、阻止魔术师接触根源，创造出了真祖；阿赖耶识为了保护人类，对抗人类生存环境里的各种威胁，创造了英灵之座。

与根源联系紧密的作品是《空之境界》；涉及到盖亚是如何运行的，是《月姬》系列；以阿赖耶识的英灵为核心的，是《Fate》系列；讲述魔术师、魔法使的故事的，是《魔法使之夜》以及贯穿所有型月作品中的魔术师故事。
这就是型月世界观的概况，以上。
